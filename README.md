# Phi 3.5

**Phi3.5**は、Microsoftが開発した論理・コード特化型のAIモデルです。小規模でありながら高度な推論能力を備え、プログラミング関連タスクに最適化されています。

## 概要

- **パラメータ数**: 14B
- **特徴**: 論理・コード特化、Microsoft製、100言語対応
- **開発元**: Microsoft
- **日本語対応**: 100言語に対応
- **コンテキスト長**: 128k トークン
- **ライセンス**: MIT

## スペック

| 項目 | 仕様 |
|------|------|
| パラメータ数 | 14B |
| メモリ要件 | 16GB+ |
| 推論速度 | 高速 |
| 言語対応 | 100言語 |
| コンテキスト | 128k トークン |
| 特化分野 | コード・論理 |

## 推奨用途

- **コード生成・レビュー**
- **プログラミング問題の解決**
- **論理的推論が必要なタスク**

- **テクニカルライティング**

## 特徴

- **コード特化**: プログラミング言語の理解と生成に最適化
- **論理的推論**: 複雑なアルゴリズム理解に強い
- **効率的**: 小規模ながら高性能
- **広言語対応**: 100言語をサポート
- **長コンテキスト**: 128k トークンで大規模コードベース対応
